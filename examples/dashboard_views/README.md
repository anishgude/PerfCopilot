# Dashboard Views

This directory holds visual artifacts that make the product easier to evaluate quickly:

- `dashboard-placeholder.svg`: placeholder until a real dashboard screenshot is captured
- `pr-comment-placeholder.svg`: placeholder until a real PR bot screenshot is captured

For internship applications, replace these placeholders with:

1. the upload/history dashboard after a successful analysis run
2. a stored run selected from history
3. a pull request comment generated by the GitHub Action
